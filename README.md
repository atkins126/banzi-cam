# banzi-cam

반디캠 흉내내기 프로젝트

## 준비 사항
- Visual Studio 2015 Update 3 또는 이후 버전
- vcpkg 설치 및 라이브러리 설치
  - ["vcpkg 설치" 참고](/install/vcpkg/)
  - vcpkg install ffmpeg portaudio boost-scope-exit
- RyuLib for C++
  - [https://github.com/ryujt/ryulib-cpp](https://github.com/ryujt/ryulib-cpp)
